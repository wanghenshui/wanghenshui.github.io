fd数目有没有上涨？

```bash
 lsof -n|awk '{print $2}'| sort | uniq -c | sort -nr | head
```



20个最高fd线程

```bash
for x in `ps -eF| awk '{ print $2 }'`;do echo `ls /proc/$x/fd 2> /dev/null | wc -l` $x `cat /proc/$x/cmdline 2> /dev/null`;done | sort -n -r | head -n 20
```





具体到进程

```bash
ll /proc/pid/fd | wc -l
```



fd都用来干啥了

```bash
strace -p pid  -f -e read,write,close
```
