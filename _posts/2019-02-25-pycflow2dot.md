##TL;DR

直接安装[pycflow2dot](https://github.com/johnyf/pycflow2dot)

```
pip install pycflow2dot
```

然后使用命令cflow2dot ，具体可以看帮助(-h)

```shell
cflow2dot -i default/berry.c -o b.svg -f svg
```

该包依赖cflow 和graphviz(aka dot)，按需安装即可
